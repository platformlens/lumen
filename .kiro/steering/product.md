# Product Overview

Lumen (formerly K8ptain) is a modern, high-performance Kubernetes management tool built as an Electron desktop application. It provides a sleek, intuitive interface for monitoring and managing Kubernetes clusters with a focus on developer experience and visual clarity.

## Core Features

- **Real-time Monitoring**: Live updates for Pods, Nodes, Deployments, and other Kubernetes resources with watchers
- **Multi-Cluster Support**: Seamless switching between different Kubernetes contexts with pinned cluster shortcuts
- **Resource Management**: Comprehensive views for workloads, networking, storage, config, and access control resources
- **Integrated Tools**: Built-in log viewer, terminal, port forwarding, and YAML editor
- **AI Assistant**: Integrated AI panel for explaining Kubernetes resources (supports Google Gemini and AWS Bedrock)
- **Visual Topology**: Resource relationship visualization and pod visualizer
- **Modern UI**: Dark-mode first design with glassmorphism aesthetics using Tailwind CSS and Framer Motion

## Target Users

DevOps engineers, platform engineers, and developers who need a powerful desktop tool for managing Kubernetes clusters with better UX than kubectl and web-based dashboards.

## Platform Support

Currently focused on macOS (arm64), with planned support for Windows and Linux.
